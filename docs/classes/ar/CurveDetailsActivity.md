# وصف الفئة: CurveDetailsActivity

________________________________________
## 1. معلومات عامة
*   **اسم الفئة:** `CurveDetailsActivity`
*   **النوع:** `Activity`
*   **الغرض:** تعرض هذه الشاشة تفاصيل منحنى محفوظ معين. تعرض الرسم مرة أخرى وتسرد جميع الحسابات (طرق مختلفة أو عدد المستطيلات) التي أجريت لهذا المنحنى.
*   **التفاعل:** 
    *   تستخدم `GraphView` في وضع القراءة فقط.
    *   تستعلم من قاعدة البيانات عن بيانات محددة تتعلق بـ `CURVE_ID`.

________________________________________
## 3. طرق الفئة (Methods)

### اسم الطريقة: `loadCurve`
*   **النوع:** `private`
*   **قيمة الإرجاع:** `void`
*   **المعلمات:** `String curveId`
*   **ماذا تفعل:** 
    1. تجد الإدخال الأول في قاعدة البيانات لهذا المعرف.
    2. تستخرج سلسلة JSON للنقاط.
    3. تستخدم **GSON** لتحويل JSON مرة أخرى إلى قائمة نقاط `PointF`.
    4. تمرر القائمة إلى `curveGraphView.setCurve()`.

________________________________________
## 8. شرح بكلمات بسيطة
هذه مثل **صفحة التفاصيل** لمشروع معين. تخيل أنك حفظت رسماً وعدة قياسات مختلفة له. تسحب هذه الشاشة ذلك الرسم من "خزانة الملفات" (قاعدة البيانات) وتظهر لك بالضبط ما رسمته وجميع نتائج الحساب التي حفظتها له سابقاً.
