# תיאור מחלקה: CurveDetailsActivity

________________________________________
## 1. מידע כללי
*   **שם המחלקה:** `CurveDetailsActivity`
*   **סוג:** `Activity`
*   **ייעוד:** מסך זה מציג את הפרטים של עקומה שמורה ספציפית. הוא מציג שוב את הציור ומפרט את כל החישובים (שיטות שונות או מספר מלבנים) שבוצעו עבור אותה עקומה.
*   **אינטראקציה:** 
    *   משתמש ב-`GraphView` במצב קריאה בלבד.
    *   שולף ממסד הנתונים נתונים ספציפיים הקשורים ל-`CURVE_ID`.

________________________________________
## 3. שיטות המחלקה

### שם השיטה: `loadCurve`
*   **סוג:** `private`
*   **ערך חזרה:** `void`
*   **פרמטרים:** `String curveId`
*   **מה היא עושה:** 
    1. מוצאת את הכניסה הראשונה בבסיס הנתונים עבור מזהה זה.
    2. מחלצת את מחרוזת ה-JSON של הנקודות.
    3. משתמשת ב-**GSON** כדי להפוך את ה-JSON בחזרה לרשימת `PointF`.
    4. מעבירה את הרשימה ל-`curveGraphView.setCurve()`.

________________________________________
## 8. הסבר במילים פשוטות
זה כמו **דף פירוט** עבור פרויקט ספציפי. תארו לעצמכם ששמרתם ציור וכמה מדידות שונות עבורו. המסך הזה שולף את הציור הזה מ"ארון התיוק" (מסד הנתונים) ומראה לכם בדיוק מה ציירתם ואת כל תוצאות החישוב ששמרתם עבורו בעבר.
